# 宠物医疗账单常见项目解释

用途：回答“账单项目是什么意思”“为什么有检查/镇静/药品/服务费”这类知识性问题，或辅助 `bill_explain` 报告。

常见类别：

- 问诊/诊疗：医生检查、咨询、复诊或急诊服务。
- 检查：血常规、生化、电解质、尿检、X 光、超声、CT、病理等。
- 镇静/麻醉：为影像、采样、处置或降低应激而使用的镇静流程和药物。
- 治疗/处置：注射、输液、清创、手术、住院护理等。
- 药品/处方粮：处方药、止痛药、抗生素、营养或处方饮食建议。
- 耗材/服务费：医疗废弃物处理、导管、针头、敷料、平台或刷卡服务费。

边界：

- 可以解释项目类别和需要向医院确认的问题。
- 不判断医院是否乱收费。
- 不能仅凭账单推断诊断或治疗必要性。
