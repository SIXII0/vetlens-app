# 宠物疑似中毒问题的紧急边界

用途：回答“吃了可能有毒的东西怎么办”这类知识性问题。

处理边界：

- 若怀疑宠物误食毒物、药物、植物、清洁剂或不明物质，应立即联系当地兽医急诊或动物毒物控制热线。
- 需要尽快保存包装、成分、摄入时间、摄入量、宠物体重和症状。
- 不在本地知识库中给出催吐、用药或等待观察指令。

美国常用资源：

- ASPCA Animal Poison Control Center: (888) 426-4435，可能收取咨询费。

边界：

- 这是紧急风险分流条目，不替代兽医或毒物控制中心建议。
